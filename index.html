rails_sequel by pusewicz

Sequel plugin for Ruby on Rails

rails_sequel allows you to quickly use Sequel Toolkit as your ORM in Ruby on Rails

Install

gem sources -a http://gems.github.com gem install pusewicz-rails_sequel
